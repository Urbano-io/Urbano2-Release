# Urbano 2 Release

[![Latest Release](https://img.shields.io/github/v/release/Urbano-io/Urbano2-Release?label=Latest%20Version&style=for-the-badge&color=blue)](https://github.com/Urbano-io/Urbano2-Release/releases/latest)

**Urbano is a new, easy-to-use, urban design tool that allows you to load urban data into Rhino.**

</div>

Enables you to build contextual models that can be used for site analysis and urban-scale mobility simulations. With v2, Urbano supports terrain and allows you to build 3D site models with different data layers and sources.
